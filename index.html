Canadian Cities Crime Analysis Tool

    Researcher:

    Project Objectives:
    This project will facilitate the research/retrieval of data, the analyses of these data, as well as visualization of the data in the form of maps. The project will retrive Canadian Census Data on various crime rate variables as per the user's choice/selections. It will then take this data, and compute tables, and other visual representation in order to compare crime rates and other variables between Canadian cities.

Data Sources:
We will require various datasets in order to complete this. Spatial Data is required in order to produce visual representations, and to geolocate our areas of interest. We also require attribute data, in order to retrive crime statistics which can then be analyzed, manipulated, and interpreted through our program.


Click on the Following Links to visit the Datasets:

Program Function / How it Will Run:

1) The program will first ask for the user to select their Canadian cities of interest

2) The program will then retrive crime data for these cities, through different variables. These variables include total crime case, total violent crime, crime rate per 100,000 people, violent crime rate per 100,000 people...etc. This retreival could be done at various scale levels (by Census Subdivisions, Dissemination Areas, Census Metropolitan Area, Census Tract...etc.)

3) Once the program has extracted the data for all the cities of interest, the user will then be able to choose which variables to include (from some of the variables mentioned in Step 2), in order to perform their comparisons/analyses.

4) The program will then take this data, and perform various comparisons between the different locations. It will be able to output tables and summary statistic comparisons.

5) The program will then output various visual representations of this process. Including a map of canada showing the selected locations, and additional crime maps (choropleth) for the selected locations.


Summary
This tool will allow someone to exmaine different cities in Canada, by looking at crime rate variables, and then comparing these various locations. How the tool can be used: In a world where Remote work is now possible, many individuals and families are seeking housing in different cities, as a result of this flexibility. With the option to work essentially anywhere, it can be hard for someone to decide where they would like to live. There are many factors to take into consideration when moving to a new city. This tool will allow them to choose various locations across Canada, and through the use of crime rate census data, they will be able to determine which cities are safest.
